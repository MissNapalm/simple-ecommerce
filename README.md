# E-commerce Portfolio Project

This is a simple e-commerce portfolio project built with vanilla JavaScript. It features a product listing page and a shopping cart functionality.

## Project Structure

```
ecommerce-portfolio
├── src
│   ├── js
│   │   ├── app.js          # Main entry point for JavaScript functionality
│   │   ├── cart.js         # Handles shopping cart functionality
│   │   └── products.js      # Manages product data
│   ├── css
│   │   └── styles.css      # Styles for the e-commerce page
│   └── data
│       └── products.json    # Contains product data
├── index.html              # Main HTML file for the e-commerce page
├── cart.html               # HTML file for displaying the shopping cart
└── README.md               # Documentation for the project
```

## Features

- Product listing with images, names, and prices.
- Ability to add products to the shopping cart.
- View cart contents and total price on a separate page.

## Setup Instructions

1. Clone the repository:
   ```
   git clone <repository-url>
   ```

2. Navigate to the project directory:
   ```
   cd ecommerce-portfolio
   ```

3. Open `index.html` in your web browser to view the e-commerce page.

## Usage

- Click on "Add to Cart" buttons to add products to your shopping cart.
- Navigate to the cart page to view your selected items and total price.

Feel free to customize the product data in `src/data/products.json` to add your own products!
